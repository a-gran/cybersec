## Шаг 2 — Написать `handlers/results.py`

### Что такое `show_results()`

Это **не хендлер** (нет декоратора `@router`). Это обычная async-функция, которую вызывает Участник 5 из `handle_answer()` когда игра заканчивается.

`router` в файле всё равно нужен — Участник 1 подключает его в `bot.py`. Просто в нём нет активных хендлеров (сейчас), только `router = Router()`.

### Что должна делать `show_results()`

1. Взять из сессии: `correct`, `score`, `lives`, `total`
2. Вызвать `final_title(score, total)` — получить звание
3. Сформировать текст итогового экрана
4. Отправить сообщение с кнопкой «Играть снова»

### Что должно быть в тексте

- Заголовок «ИТОГ ИГРЫ»
- Правильных ответов: X из Y
- Итоговый счёт в очках
- Оставшиеся жизни (можно визуально: ❤️ ❤️ 🖤)
- Звание из `final_title()`
- Список признаков скама из `SCAM_SIGNS`

### Пример того как должно выглядеть в Telegram

```
━━━━━━━━━━━━━━━━━━━━━━
        🏁 ИТОГ ИГРЫ
━━━━━━━━━━━━━━━━━━━━━━

✅ Правильных ответов: 14 из 20
⭐ Итоговый счёт:      140 очков
❤️ Оставшиеся жизни:   ❤️ ❤️ ❤️

🏆 КИБЕРДЕТЕКТИВ — отличный результат!

🔍 Главные признаки скама:
  • просят пароль или код подтверждения
  • слишком выгодное предложение (бесплатный приз)
  • искусственная срочность (только сейчас!)
  • подозрительные ссылки на незнакомые сайты
  • просят данные карты или CVV

[🔄 Играть снова]
```

### Пример реализации

```python
from aiogram import Router
from aiogram.types import Message

from config import SCAM_SIGNS
from game_logic.answer import final_title
from keyboards import play_again_keyboard

router = Router()


async def show_results(message: Message, session: dict):
    correct = session["correct"]
    score   = session["score"]
    lives   = session["lives"]
    total   = session["total"]

    title = final_title(score, total)

    lives_display = "❤️ " * lives + "🖤 " * (3 - lives)
    signs = "\n".join(f"  • {sign}" for sign in SCAM_SIGNS)

    result_text = (
        "━━━━━━━━━━━━━━━━━━━━━━\n"
        "        🏁 <b>ИТОГ ИГРЫ</b>\n"
        "━━━━━━━━━━━━━━━━━━━━━━\n\n"
        f"✅ Правильных ответов: <b>{correct} из {total}</b>\n"
        f"⭐ Итоговый счёт:      <b>{score} очков</b>\n"
        f"❤️ Оставшиеся жизни:   {lives_display}\n\n"
        f"{title}\n\n"
        "🔍 <b>Главные признаки скама:</b>\n"
        f"{signs}"
    )

    await message.answer(result_text, parse_mode="HTML", reply_markup=play_again_keyboard())
```

## Шаг 4 — Тестирование всего бота (основная задача)

Когда все участники сдали свои части и Участник 1 запустил бота — ты проводишь финальное тестирование.

### Как тестировать

Открой Telegram, найди бота и пройди каждый сценарий из таблицы ниже. Записывай результат: ✅ работает / ❌ ошибка (и что именно).

### Чеклист тестирования

#### Сценарий 1 — Базовый флоу

| # | Действие | Ожидаемый результат |
|---|---|---|
| 1.1 | Написать `/start` | Появляются правила и кнопка «Начать игру» |
| 1.2 | Нажать «Начать игру» | Кнопка исчезает, появляется первый вопрос с кнопками |
| 1.3 | Нажать «Скам» или «Безопасно» | Кнопки исчезают, появляется «ПРАВИЛЬНО» или «НЕПРАВИЛЬНО» с объяснением |
| 1.4 | Дождаться следующего вопроса | Автоматически приходит вопрос 2/20 |
